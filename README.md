# jable.tv多线程视频下载工具
how to use?

    1,首先必备代理服务器，请修改proxy_url 配置代理服务器

    2,修改url视频链接  注意结尾 / 不要漏掉了

    3,配置线程池数量，请根据实际配置填写 默认 thread_nums = 20

    pip install pycryptodome --upgrade
    
    运行 python main.py
